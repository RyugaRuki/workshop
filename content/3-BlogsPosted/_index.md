### [Blog 1 - Improving Order History Search using Semantic Search with Amazon OpenSearch Service](3.1-Blog1/)

This blog explains how Amazon enhanced the "Your Orders" search experience by integrating semantic search with its existing lexical search solution. It discusses the use of Amazon OpenSearch Service and Amazon SageMaker to build a hybrid search system, along with the resilient cell-based architecture that improves search relevance and system reliability.

### [Blog 2 - Simplify AWS AppSync Events Integration with Powertools for AWS Lambda](3.2-Blog2/)

This blog introduces the AppSyncEventsResolver feature in Powertools for AWS Lambda, demonstrating how developers can efficiently process real-time events from AWS AppSync. It covers flexible event routing, automatic response formatting, batch error handling, and techniques to reduce boilerplate code in Serverless applications.

### [Blog 3 - Amazon S3 Audit Logging](3.3-Blog3/)

This blog focuses on identity-based security auditing for Amazon S3 using AWS CloudTrail data events and Amazon Athena. It explains how to configure organization-wide audit logging, optimize Athena tables with partition projection, and perform security investigations by querying user activities, cross-account access, and failed operations.
